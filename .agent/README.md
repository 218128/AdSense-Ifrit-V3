# AdSense Ifrit - AI Agent Onboarding

**Purpose**: This folder contains structured documentation to help AI coding agents understand and work effectively on this project.

## Quick Start for AI Agents

1. Read `ARCHITECTURE.md` - Understand the system design
2. Read `API_REFERENCE.md` - Know the API endpoints
3. Read `DATA_MODELS.md` - Understand the data structures
4. Check `/CONTRIBUTING.md` - Follow coding conventions

## Key Entry Points

| Area | Path | Description |
|------|------|-------------|
| App Routes | `app/` | Next.js 16 App Router pages and API |
| Components | `components/` | React UI components |
| Libraries | `lib/` | Core business logic |
| Websites | `websites/{domain}/` | Per-website data storage |
| Templates | `lib/templates/` | Site generation templates |

## Common Tasks

- **Add API endpoint**: Create `app/api/path/route.ts`
- **Add component**: Create in `components/{feature}/Name.tsx`
- **Add utility**: Create in `lib/{category}/feature.ts`
- **Test a website**: Check `websites/{domain}/` folder
